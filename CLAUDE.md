# Claude Code Mobile Bridge

## /sync — Підтягнути зміни з мобільної сесії

Коли Лука каже "sync", "/sync", "підтягни зміни", "що було в терміналі" — **одразу без питань** виконай:

```bash
cd "<workspace-root>" && ./scripts/sync.sh
```

Прочитай весь вивід і **підсумуй** що відбулось в мобільній (CLI) сесії. Зверни увагу на:
- Які задачі обговорювались
- Які рішення прийняті
- Які файли змінені
- Чи є незавершені задачі

Не питай "хочеш підтягнути?" — просто роби.
